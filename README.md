# HProxie
Bring your own proxy to your browser.

This extention only works for http/s proxies.
# Installation
- Un-zip the file
- Go to Manage extentions [Edge - edge://extensions/] [Chrome - chrome://extensions]
- Enable Developer Mode
- Load Unpacked
- Select the Un-zipped Folder
- DONE!
